Checklist Item Descriptions

Update Name

Update the program name in the system to reflect any changes or corrections needed.

Discontinuation Date

Set or update the date when this program will no longer be offered.

Program Prices

Review and update the pricing structure for the entire program.

Duplicate Offering

Create a copy of an existing offering to use as a template for the new semester.

Offering Name

Update or set the name for this specific offering instance.

Semester

Assign the appropriate semester/term designation (e.g., Fall 2025, Spring 2026).

Age Restrictions

Set or update any age requirements or restrictions for participants in this offering.

Registration Open and Close

Set the dates when registration opens and closes for this offering.

Offering Prices

Review and update the pricing for this specific offering instance.

Input Schedule

Enter the dates, times, and locations for all sessions in this offering.

Remove Canceled Dates

Delete any dates that have been canceled from the schedule (e.g., holidays, weather cancellations).
